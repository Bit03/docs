    开发环境

    基本环境

    • 系统环境: Ubuntu 16.04 LTS
    • 数据库: MySQL 5.6
    • 缓存系统: redis
    • 开发语言: Python 3.6.x
    • 搜索引擎: elasticsearch 2.4.6
    • 图片系统使用: 七牛云

    Web Framework

    • Django-1.11.x
    • django-rest-framework 3.9.x

    前端

    • webpack-4.35.3
    • spectre
    cd static/
npm install
npm run dev

    GeoIP DB

    cd project_dir/
python manage.py update_mmdb
